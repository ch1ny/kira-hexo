# 文章归档

## 用法

在 hexo 项目根目录的 `source` 文件夹下新建一个 Markdown 文件，文件名随意。

> 比如我自己的站点就是叫做 `archive.md`。

然后在文件中填入：
```md
---
title: 文章归档
layout: archives
---
```

:::tip
其中的 `title` 字段可以根据自己的需要替换成自己喜欢的页面标题，但是 `layout` 的值**必须是** `archives`。
:::
