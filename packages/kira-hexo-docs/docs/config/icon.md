## 新增图标
如果需要自己添加图标的话，建议使用[阿里巴巴矢量图标库](https://www.iconfont.cn/)：

1. 注册登录阿里巴巴矢量图标库
2. [资源管理 - 我的项目](https://www.iconfont.cn/manage/index?manage_type=myprojects)
3. 新建项目，`Font Family` 参数设置为 `kirafont`
4. 添加新图标至项目
5. 可以直接选择生成 CDN 地址使用，但是建议打包资源到本地使用
